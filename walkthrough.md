# Portfolio Website Walkthrough

I have successfully created a multi-page, minimalist portfolio website for Harsh Ringsia.

## Features

- **Minimalist Design**: Black and white theme with clean typography (Inter).
- **Dark/Light Mode**: Toggleable theme with persistent state.
- **Custom Cursor**: Animated cursor with hover and click effects.
- **Responsive Layout**: Fully responsive design for mobile and desktop.
- **Pages**:
    - **Home**: Hero section with name and tagline.
    - **About**: Education, Skills, and Certifications.
    - **Experience**: Detailed work history.
    - **Projects**: Showcase of projects with descriptions and tags.
    - **Contact**: Contact information and social links.

## Tech Stack

- **React (Vite)**: Fast and modern frontend build tool.
- **Framer Motion**: Smooth animations for page transitions and cursor.
- **Lucide React**: Consistent and clean icons.
- **React Router DOM**: Client-side routing.

## Verification

- **Build**: The project builds successfully with `npm run build`.
- **Linting**: Code structure follows best practices.
- **Content**: All resume details have been integrated into the respective pages.

## How to Run

1.  Install dependencies: `npm install`
2.  Start dev server: `npm run dev`
3.  Build for production: `npm run build`
